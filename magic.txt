✨ - Hello, baby! precisa de ajuda na instalação via termux?
relaxa, aqui está o passo a passo 
━━━━━━━━━━━━━━━━━━━━━━━
⚙️ 1º Comando (ajustar repositórios):  

termux-change-repo  

⚙️ 2º Comando (atualizar pacotes):
 
apt-get upgrade -y  

⚙️ 3º Comando (atualizar sistema):
 
apt-get update -y  

⚙️ 4º Comando (instalar dependências principais):

pkg upgrade -y && pkg update -y && pkg install nodejs -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && pkg install wget -y && pkg install tesseract -y && pkg install git -y  

⚙️ 5º Comando (permitir acesso ao armazenamento):

termux-setup-storage  

⚙️ 6º Comando (entrar na pasta do bot):
 
cd /sdcard/Gwen-Bot

⚙️ 7º Comando (iniciar o bot):

sh start.sh
